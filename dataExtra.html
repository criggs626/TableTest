Data-Extra

The Methods listed below are extra methods that are either similar to another fully documented method or short enough to not warrant a drawn out explanation

Data-Uri

Data-Uri simply returns the uri of the item clicked

return ajax.uri(this, event);

It then sets up listeners to see what action is picked if cancel it closes the modal. If confirm it navigates to the specified href attribute.

Data-Form

Data-Form simply takes a form that is being submitted and submits it as an AJAX

return ajax.submit(this, event);

Data-Form-Confirm

Data-Form-Confirm Does similar actions as Data-Confirm but instead of navigating on confirm it submits a form.

$("input[name='confirm']", scope).click(function () {
    ajax.submit($this, $event);
});

Data-Form-Confirm-Post

Data-Form-Confirm Does similar actions as Data-Confirm and Data-Form-Confirm but on confirm submits the form in a different manner.

$("input[name='confirm']", scope).click(function () {
    $($this).data("cont", true);
    $this.submit();
});
Note: For More information on any of the data confirm methods go to Data-Confirm
