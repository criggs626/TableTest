Table-Navigation

This describes how to add navigation that will work with your table.

After you've set up your table a navigation class is automatically started that listens to certain elements. These elements are placed in a form and each given the class nav so the navigation function can listen to a $(".nav").on("click",function{...}) and id's to tell the listener what action to take.

The Id's and their actions are listed below

  • ID:first navigates to the first page
  • ID:prev navigates to the previous page
  • ID:next navigates to the next page
  • ID:last navigates to the last page
  • ID:size is a select menu with different size options for pagination. Each item has to be given the correct size value
  • ID:all is the last item in the select menu and it's value is set programmatically to show all items
  • ID:pageNumber is where the current page number out of the total page numbers is displayed
