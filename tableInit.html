Table-Initialize

Initializing the table allows you to add content from JSON to a table along with adding pagination, sorting, and searching.

The first step is to set up the table, you do this by adding items to that corespond to your data in the <thead> tag. And specifying sort types for each column. You also have to give the table an id and add an empty body tag

Supported types include int, string, and date. To add more types look up StupidTable Creating your own data type
Classes used are from Bootstrap. 

<table id='myTable' class='table table-condensed table-striped'>
    <thead>
        <tr>
            <th data-sort="string">Name</th>      <!-- This column is sortable by string-->
            <th>Status</th>                       <!-- This column is not sortable -->
            <th data-sort="date">Birth Date</th>  <!-- This column is sortable by date -->
            <th data-sort="int">Age</th>          <!-- This column is sortable by int -->
        </tr>
    </thead>
    <tbody></tbody>
</table>

The initializer is the command table(String tableId, int itemsPerPage, JSON tableContent);

Note: when using JSON the table data is stored as a two dimensional matrix in the data attribute. Along with having a records total attribute with the total number of records.

table("myTable", 5, content);
//myTable is the table Id above
//5 is the starting value for items per page
//content is a variable with the value of our JSON to be added to the table
