Data-Modal

The data-modal handler is built for items that will open a modal window.

The field begins by getting the information required for a query from data attributes on the input type. 

var settings = $(this).data("modal");
var href = $(this).attr("href");
var title = $(this).attr("title");
var callback = $(this).data("callback");
var container = "";

The block then checks the settings and reads the handler to determine what action to take.

if (settings.x.indexOf("%") != -1) {...}
if (settings.handler == "iframe" && href) {...}
if (settings.handler == "ajax" && href) {...}
if (settings.handler == "img" && href) {...}

The actual button is setup like this.

<button data-modal='{"handler":"img","y":"30%","x":"30%"}'href='img/icon.png'>
Open Modal</button>

The modal should be setup like this.

<div id="modal" class="modal">
    <div id="modal-topbox"> </div>
    <button class="btn btn-danger" onclick='$("#modal").hide()'>X</button>
    <div id="modal-container">
    </div>
</div>
