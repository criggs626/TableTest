Table-Update

Updating the table allows you to add content from JSON to an existing table without removing previous content

After you've setup the table you simply have to call tableUpdate(JSON content) and it will add the content to the table

table("myTable", 10, content);
//myTable is the table Id above
//5 is the starting value for items per page
//content is a variable with the value of our JSON to be added to the table
function update(){
    tableUpdate(updateContent)
    //Where updateContent is the updated content
}
